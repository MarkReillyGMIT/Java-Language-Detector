Object Oriented Programming Multithreaded Language Detector
A Java API that can rapidly compare a query text file against a ngram collection of subject texts and determine the natural language of the query file.

Features:
- Detects language of the text file
- Multithreaded API
- Allows user to enter the dataset and query text files.
- Allows user to choose the size of each n-gram.
- JavaDocs
- UML Diagram

User Guide:
- The program can be run by using the following command in Command Line:
	java -cp "Path to the oop.jar" ie.gmit.sw.Runner
- The program can also be run in eclipse on the console.
